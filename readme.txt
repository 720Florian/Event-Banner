Display scheduled or manually activated event banners with optional links.

== Description ==
Event Banner lets you create banners for announcements or events, schedule them by date/time, or activate them manually. You can optionally link to a post or URL, and choose where the banner appears (top of page, footer, or shortcode only).

== Installation ==
1. Upload the `event-banner` folder to `/wp-content/plugins/`.
2. Activate the plugin via the Plugins menu in WordPress.
3. Go to Settings → Event Banner to configure display options.
4. Create banners under Event Banners in the admin menu.

== Frequently Asked Questions ==

= How do I display the banner? =
Use automatic display (Settings → Event Banner), or add the `[event_banner]` shortcode to a page/template.

= Can I schedule banners? =
Yes. Set start and end date/time (24h format), or activate manually.
